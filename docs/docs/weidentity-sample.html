WeIdentity Sample 开发样例¶

weid-sample 是基于 WeIdentity 开发的 Java 应用样例程序，提供了一整套的流程演示，可以帮您快速理解 WeIdentity 的运行机制，您也可以参考该样例程序，开发您的 Java 应用。

环境准备¶

您可以参考 WeIdentity JAVA SDK 安装部署文档中的准备工作章节进行准备。

快速体验¶

我们提供了两种方式体验 weid-sample：

  • 命令行方式使用 （推荐方式）

  • spring-boot方式使用
